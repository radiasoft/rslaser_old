GaussianBlurBlobs.py applies a simple gaussian filter to the provided images and produces plots with
various gaussian sigmas. It can be run with

  python3 GaussianBlurBlobs.py <filepath>


SuperGaussianFitting.py fits a "super gaussian" function to a provided image, and plots the result along with the
contours of the image. It has not been tested on a large variety of images and may timeout or produce sub-par results

  python3 SuperGaussianFitting.py <filepath>
